## 🗺️ 第一層（1F）の探索と目標

序盤の目標は、探索を通じて資金を集め、装備を整えることです。

| 目標 | 詳細 |
| :--- | :--- |
| **資金稼ぎ** | 敵を倒してドロップ品や不要なアイテムを集めましょう。 |
| **装備の購入** | トレーダーでお金を使い、より強力な武器や装備に更新しましょう。 |
| **次の階層へ** | 第二層へ通じる入口を発見し、冒険の幅を広げましょう。 |

## ✨ スキルの割り当てと発動

スキルを獲得したら、戦闘で使用するために以下の準備が必要です。

<Steps>

1. **スキルの割り当て**
   `/skills` コマンドで管理画面を開き、獲得したスキルをショートカットに割り当てます。

2. **発動モードへの切り替え**
   **オフハンド切り替えキー**（デフォルト設定では `F` キー）を押して、スキル発動モードに入ります。

3. **スキル発動**
   スキル発動モード中に対応するスロットのキーを押すことで、スキルが発動します。

</Steps>
